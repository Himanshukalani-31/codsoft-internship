This repository contains the projects completed during my 1-Month Virtual Internship as a Python Programmer at CodSoft (Duration: 05 June 2026 to 05 July 2026).

The internship focused on developing Command-Line Interface (CLI) applications, strengthening core Python concepts, implementing Object-Oriented Programming (OOP), and applying effective error-handling techniques.

📁 Repository Structure & Task Overview

The following are the five projects completed during the internship:

| Task Number | Project Name              | Description                                                  | Key Features                                                                                                                                                 |
| ----------- | -----------------------   | ------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Task 1      | Advanced To-Do List       | A task management application with structured data handling. | Dictionary-based storage, automatic unique ID generation, status tracking ([✓]/[ ]), and complete CRUD (Create, Read, Update, Delete) operations.            |
